# To-Do Daily tasks Application

## Description
A simple Python project to manage daily tasks. Users can add, view, and delete tasks.

## Features
- Add new tasks
- View all tasks
- Delete completed tasks
- Simple console-based interface

## Technologies Used
- Python

## How to Run
1. Open the todo_list.py file in Python IDE.
2. Run the program.
3. Use the menu to add, view, or remove tasks.
